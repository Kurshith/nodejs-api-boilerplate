## Nodejs api boilerplate

This is a minimalistic Nodejs API boilerplate

## Uses

1. `Mongodb 3.6.3`
2. `Express 4.13.3`
3. `Nodejs v8.9.4`

## Deployment

1. Set the environment variable `MY_SECRET` for use when generating JWT token

2. Install the dependencies using `npm install`
